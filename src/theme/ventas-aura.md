# Ventas Aura - Guía de Uso del Tema

1. **Jerarquía Visual**: Utiliza `text-primary` para los datos clave y `text-secondary` con opacidad para el ruido visual y metadatos.
2. **Glassmorphism Inteligente**: Aplica `crystal-card` solo a elementos elevados (tarjetas, modales), manteniendo el fondo base limpio y minimalista.
3. **Acentos y Color**: El gradiente primario y colores semánticos (tags al 15% opacity) deben usarse únicamente en llamadas a la acción (CTAs) y estados, no para decorar.
